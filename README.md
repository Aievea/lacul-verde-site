  # Lacul Verde Sâncrai                                                                                                                                                          
  
  Website oficial al bălții de pescuit **Lacul Verde**, situată în Sâncraiu de Mureș, județul Mureș, România.

  ## Pagini

  - **Acasă** — prezentare generală, specii de pești și atmosfera locului
  - **Specii Pești** — galerie detaliată cu speciile din lac
  - **Tarife** — prețuri actualizate, taxe pe zi / jumătate de zi, prețuri pește
  - **Regulament** — regulile oficiale ale bălții
  - **Contact** — locație pe hartă, WhatsApp, Facebook

  ## Tech Stack

  - [Next.js 16](https://nextjs.org/)
  - [Tailwind CSS v4](https://tailwindcss.com/)
  - [Framer Motion](https://www.framer.com/motion/)
  - [Lucide React](https://lucide.dev/)

  ## Rulare locală

  npm install
  npm run dev
